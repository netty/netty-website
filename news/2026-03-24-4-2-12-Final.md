We are happy to announce the release of netty 4.2.12.Final. This is a bug-fix release that fixes a regression introduced in 4.2.11.Final.

We strongly recommend upgrading to this version to get the security fixes included in 4.2.11.Final.

The security fixes are:

* [CVE-2026-33871](https://github.com/netty/netty/security/advisories/GHSA-w9fj-cfpg-grvv) rated _high_. This is an HTTP/2 CONTINUATION frame flood Denial of Service vulnerability.
* [CVE-2026-33870](https://github.com/netty/netty/security/advisories/GHSA-pwqr-wmgm-9rr8) rated _high_. This is an HTTP/1.1 Request Smuggling vulnerability in chunked encoding parsing.

The regerssion fix is:

* Revert "Eliminate redundant bounds checks in CompositeByteBuf accessors" by @chrisvest in https://github.com/netty/netty/pull/16550

For more details please see the complete release notes: https://github.com/netty/netty/releases/tag/netty-4.2.12.Final
