44  /**
45   * <p><em>This API is very immature.</em> The Http2Connection-based API is currently preferred over this API.
46   * This API is targeted to eventually replace or reduce the need for the {@link Http2ConnectionHandler} API.
47   *
48   * <p>An HTTP/2 handler that maps HTTP/2 frames to {@link Http2Frame} objects and vice versa. For every incoming HTTP/2
49   * frame, an {@link Http2Frame} object is created and propagated via {@link #channelRead}. Outbound {@link Http2Frame}
50   * objects received via {@link #write} are converted to the HTTP/2 wire format. HTTP/2 frames specific to a stream
51   * implement the {@link Http2StreamFrame} interface. The {@link Http2FrameCodec} is instantiated using the
52   * {@link Http2FrameCodecBuilder}. It's recommended for channel handlers to inherit from the
53   * {@link Http2ChannelDuplexHandler}, as it provides additional functionality like iterating over all active streams or
54   * creating outbound streams.
55   *
56   * <h3>Stream Lifecycle</h3>
57   * <p>
58   * The frame codec delivers and writes frames for active streams. An active stream is closed when either side sends a
59   * {@code RST_STREAM} frame or both sides send a frame with the {@code END_STREAM} flag set. Each
60   * {@link Http2StreamFrame} has a {@link Http2FrameStream} object attached that uniquely identifies a particular stream.
61   *
62   * <p>{@link Http2StreamFrame}s read from the channel always a {@link Http2FrameStream} object set, while when writing a
63   * {@link Http2StreamFrame} the application code needs to set a {@link Http2FrameStream} object using
64   * {@link Http2StreamFrame#stream(Http2FrameStream)}.
65   *
66   * <h3>Flow control</h3>
67   * <p>
68   * The frame codec automatically increments stream and connection flow control windows.
69   *
70   * <p>Incoming flow controlled frames need to be consumed by writing a {@link Http2WindowUpdateFrame} with the consumed
71   * number of bytes and the corresponding stream identifier set to the frame codec.
72   *
73   * <p>The local stream-level flow control window can be changed by writing a {@link Http2SettingsFrame} with the
74   * {@link Http2Settings#initialWindowSize()} set to the targeted value.
75   *
76   * <p>The connection-level flow control window can be changed by writing a {@link Http2WindowUpdateFrame} with the
77   * desired window size <em>increment</em> in bytes and the stream identifier set to {@code 0}. By default the initial
78   * connection-level flow control window is the same as initial stream-level flow control window.
79   *
80   * <h3>New inbound Streams</h3>
81   * <p>
82   * The first frame of an HTTP/2 stream must be an {@link Http2HeadersFrame}, which will have an {@link Http2FrameStream}
83   * object attached.
84   *
85   * <h3>New outbound Streams</h3>
86   * <p>
87   * A outbound HTTP/2 stream can be created by first instantiating a new {@link Http2FrameStream} object via
88   * {@link Http2ChannelDuplexHandler#newStream()}, and then writing a {@link Http2HeadersFrame} object with the stream
89   * attached.
90   *
91   * <pre>{@code
92   *     final Http2Stream2 stream = handler.newStream();
93   *     ctx.write(headersFrame.stream(stream)).addListener(new FutureListener<Void>() {
94   *
95   *         @Override
96   *         public void operationComplete(Future<Void> f) {
97   *             if (f.isSuccess()) {
98   *                 // Stream is active and stream.id() returns a valid stream identifier.
99   *                 System.out.println("New stream with id " + stream.id() + " created.");
100  *             } else {
101  *                 // Stream failed to become active. Handle error.
102  *                 if (f.cause() instanceof Http2NoMoreStreamIdsException) {
103  *
104  *                 } else if (f.cause() instanceof Http2GoAwayException) {
105  *
106  *                 } else {
107  *
108  *                 }
109  *             }
110  *         }
111  *     }
112  * }</pre>
113  *
114  * <p>If a new stream cannot be created due to stream id exhaustion of the endpoint, the {@link Promise} of the
115  * HEADERS frame will fail with a {@link Http2NoMoreStreamIdsException}.
116  *
117  * <p>The HTTP/2 standard allows for an endpoint to limit the maximum number of concurrently active streams via the
118  * {@code SETTINGS_MAX_CONCURRENT_STREAMS} setting. When this limit is reached, no new streams can be created. However,
119  * the {@link Http2FrameCodec} can be build with
120  * {@link Http2FrameCodecBuilder#encoderEnforceMaxConcurrentStreams(boolean)} enabled, in which case a new stream and
121  * its associated frames will be buffered until either the limit is increased or an active stream is closed. It's,
122  * however, possible that a buffered stream will never become active. That is, the channel might
123  * get closed or a GO_AWAY frame might be received. In the first case, all writes of buffered streams will fail with a
124  * {@link Http2ChannelClosedException}. In the second case, all writes of buffered streams with an identifier less than
125  * the last stream identifier of the GO_AWAY frame will fail with a {@link Http2GoAwayException}.
126  *
127  * <h3>Error Handling</h3>
128  * <p>
129  * Exceptions and errors are propagated via {@link ChannelHandler#exceptionCaught}. Exceptions that apply to
130  * a specific HTTP/2 stream are wrapped in a {@link Http2FrameStreamException} and have the corresponding
131  * {@link Http2FrameStream} object attached.
132  *
133  * <h3>Reference Counting</h3>
134  * <p>
135  * Some {@link Http2StreamFrame}s implement the {@link ReferenceCounted} interface, as they carry
136  * reference counted objects (e.g. {@link ByteBuf}s). The frame codec will call {@link ReferenceCounted#retain()} before
137  * propagating a reference counted object through the pipeline, and thus an application handler needs to release such
138  * an object after having consumed it. For more information on reference counting take a look at
139  * https://netty.io/wiki/reference-counted-objects.html
140  *
141  * <h3>HTTP Upgrade</h3>
142  * <p>
143  * Server-side HTTP to HTTP/2 upgrade is supported in conjunction with {@link Http2ServerUpgradeCodec}; the necessary
144  * HTTP-to-HTTP/2 conversion is performed automatically.
145  */

185      * Iterates over all active HTTP/2 streams.
186      *
187      * <p>This method must not be called outside of the event loop.
188      */
189     final void forEachActiveStream(final Http2FrameStreamVisitor streamVisitor) throws Http2Exception {
190         assert ctx.executor().inEventLoop();
191 
192         if (connection().numActiveStreams() > 0) {
193             connection().forEachActiveStream(stream -> {
194                 try {
195                     return streamVisitor.visit(stream.getProperty(streamKey));
196                 } catch (Throwable cause) {
197                     onError(ctx, false, cause);
198                     return false;
199                 }
200             });
201         }

225     private void tryExpandConnectionFlowControlWindow(Http2Connection connection) throws Http2Exception {
226         if (initialFlowControlWindowSize != null) {
227             // The window size in the settings explicitly excludes the connection window. So we manually manipulate the
228             // connection window to accommodate more concurrent data per connection.
229             Http2Stream connectionStream = connection.connectionStream();
230             Http2LocalFlowController localFlowController = connection.local().flowController();
231             final int delta = initialFlowControlWindowSize - localFlowController.initialWindowSize(connectionStream);
232             // Only increase the connection window, don't decrease it.
233             if (delta > 0) {
234                 // Double the delta just so a single stream can't exhaust the connection window.
235                 localFlowController.incrementWindowSize(connectionStream, Math.max(delta << 1, delta));
236                 flush(ctx);
237             }
238         }
239     }

241     /**
242      * Handles the cleartext HTTP upgrade event. If an upgrade occurred, sends a simple response via
243      * HTTP/2 on stream 1 (the stream specifically reserved for cleartext HTTP upgrade).
244      */
245     @Override
246     public final void userEventTriggered(final ChannelHandlerContext ctx, final Object evt) throws Exception {
247         if (evt == Http2ConnectionPrefaceAndSettingsFrameWrittenEvent.INSTANCE) {
248             // The user event implies that we are on the client.
249             tryExpandConnectionFlowControlWindow(connection());
250 
251             // We schedule this on the EventExecutor to allow to have any extra handlers added to the pipeline
252             // before we pass the event to the next handler. This is needed as the event may be called from within
253             // handlerAdded(...) which will be run before other handlers will be added to the pipeline.
254             ctx.executor().execute(() -> ctx.fireUserEventTriggered(evt));
255         } else if (evt instanceof UpgradeEvent) {
256             try (UpgradeEvent upgrade = (UpgradeEvent) evt) {
257                 ctx.fireUserEventTriggered(evt);
258                 Http2Stream stream = connection().stream(HTTP_UPGRADE_STREAM_ID);
259                 if (stream.getProperty(streamKey) == null) {
260                     // TODO: improve handler/stream lifecycle so that stream isn't active before handler added.
261                     // The stream was already made active, but ctx may have been null so it wasn't initialized.
262                     // https://github.com/netty/netty/issues/4942
263                     onStreamActive0(stream);
264                 }
265                 upgrade.upgradeRequest().headers().setInt(
266                         HttpConversionUtil.ExtensionHeaderNames.STREAM_ID.text(), HTTP_UPGRADE_STREAM_ID);
267                 stream.setProperty(upgradeKey, true);
268                 InboundHttpToHttp2Adapter.handle(
269                         ctx, connection(), decoder().frameListener(), upgrade.upgradeRequest());
270             }
271         } else {
272             ctx.fireUserEventTriggered(evt);
273         }
274     }

276     /**
277      * Processes all {@link Http2Frame}s. {@link Http2StreamFrame}s may only originate in child
278      * streams.
279      */
280     @Override
281     public Future<Void> write(ChannelHandlerContext ctx, Object msg) {
282         if (msg instanceof Http2DataFrame) {
283             Http2DataFrame dataFrame = (Http2DataFrame) msg;
284             return encoder().writeData(ctx, dataFrame.stream().id(), dataFrame.content(),
285                     dataFrame.padding(), dataFrame.isEndStream());
286         } else if (msg instanceof Http2HeadersFrame) {
287             return writeHeadersFrame(ctx, (Http2HeadersFrame) msg);
288         } else if (msg instanceof Http2WindowUpdateFrame) {
289             Http2WindowUpdateFrame frame = (Http2WindowUpdateFrame) msg;
290             Http2FrameStream frameStream = frame.stream();
291             // It is legit to send a WINDOW_UPDATE frame for the connection stream. The parent channel doesn't attempt
292             // to set the Http2FrameStream so we assume if it is null the WINDOW_UPDATE is for the connection stream.
293             try {
294                 if (frameStream == null) {
295                     increaseInitialConnectionWindow(frame.windowSizeIncrement());
296                 } else {
297                     consumeBytes(frameStream.id(), frame.windowSizeIncrement());
298                 }
299                 return ctx.newSucceededFuture();
300             } catch (Throwable t) {
301                 return ctx.newFailedFuture(t);
302             }
303         } else if (msg instanceof Http2ResetFrame) {
304             Http2ResetFrame rstFrame = (Http2ResetFrame) msg;
305             int id = rstFrame.stream().id();
306             // Only ever send a reset frame if stream may have existed before as otherwise we may send a RST on a
307             // stream in an invalid state and cause a connection error.
308             if (connection().streamMayHaveExisted(id)) {
309                 return encoder().writeRstStream(ctx, rstFrame.stream().id(), rstFrame.errorCode());
310             } else {
311                 ReferenceCountUtil.release(rstFrame);
312                 return ctx.newFailedFuture(Http2Exception.streamError(
313                         rstFrame.stream().id(), Http2Error.PROTOCOL_ERROR, "Stream never existed"));
314             }
315         } else if (msg instanceof Http2PingFrame) {
316             Http2PingFrame frame = (Http2PingFrame) msg;
317             return encoder().writePing(ctx, frame.ack(), frame.content());
318         } else if (msg instanceof Http2SettingsFrame) {
319             return encoder().writeSettings(ctx, ((Http2SettingsFrame) msg).settings());
320         } else if (msg instanceof Http2SettingsAckFrame) {
321             // In the event of manual SETTINGS ACK, it is assumed the encoder will apply the earliest received but not
322             // yet ACKed settings.
323             return encoder().writeSettingsAck(ctx);
324         } else if (msg instanceof Http2GoAwayFrame) {
325             return writeGoAwayFrame(ctx, (Http2GoAwayFrame) msg);
326         } else if (msg instanceof Http2PushPromiseFrame) {
327             Http2PushPromiseFrame pushPromiseFrame = (Http2PushPromiseFrame) msg;
328             return writePushPromise(ctx, pushPromiseFrame);
329         } else if (msg instanceof Http2PriorityFrame) {
330             Http2PriorityFrame priorityFrame = (Http2PriorityFrame) msg;
331             return encoder().writePriority(ctx, priorityFrame.stream().id(), priorityFrame.streamDependency(),
332                     priorityFrame.weight(), priorityFrame.exclusive());
333         } else if (msg instanceof Http2UnknownFrame) {
334             Http2UnknownFrame unknownFrame = (Http2UnknownFrame) msg;
335             return encoder().writeFrame(ctx, unknownFrame.frameType(), unknownFrame.stream().id(),
336                     unknownFrame.flags(), unknownFrame.content());
337         } else if (!(msg instanceof Http2Frame)) {
338             return ctx.write(msg);
339         } else {
340             ReferenceCountUtil.release(msg);
341             return ctx.newFailedFuture(new UnsupportedMessageTypeException(msg));
342         }
343     }

345     private void increaseInitialConnectionWindow(int deltaBytes) throws Http2Exception {
346         // The LocalFlowController is responsible for detecting over/under flow.
347         connection().local().flowController().incrementWindowSize(connection().connectionStream(), deltaBytes);
348     }

350     final boolean consumeBytes(int streamId, int bytes) throws Http2Exception {
351         Http2Stream stream = connection().stream(streamId);
352         // Upgraded requests are ineligible for stream control. We add the null check
353         // in case the stream has been deregistered.
354         if (stream != null && streamId == HTTP_UPGRADE_STREAM_ID) {
355             Boolean upgraded = stream.getProperty(upgradeKey);
356             if (Boolean.TRUE.equals(upgraded)) {
357                 return false;
358             }
359         }
360 
361         return connection().local().flowController().consumeBytes(stream, bytes);
362     }

364     private Future<Void> writeGoAwayFrame(ChannelHandlerContext ctx, Http2GoAwayFrame frame) {
365         if (frame.lastStreamId() > -1) {
366             frame.release();
367             return ctx.newFailedFuture(new IllegalArgumentException("Last stream id must not be set on GOAWAY frame"));
368         }
369 
370         int lastStreamCreated = connection().remote().lastStreamCreated();
371         long lastStreamId = lastStreamCreated + ((long) frame.extraStreamIds()) * 2;
372         // Check if the computation overflowed.
373         if (lastStreamId > Integer.MAX_VALUE) {
374             lastStreamId = Integer.MAX_VALUE;
375         }
376         return goAway(ctx, (int) lastStreamId, frame.errorCode(), frame.content());
377     }

379     private Future<Void> writeHeadersFrame(final ChannelHandlerContext ctx, Http2HeadersFrame headersFrame) {
380 
381         if (isStreamIdValid(headersFrame.stream().id())) {
382             return encoder().writeHeaders(ctx, headersFrame.stream().id(), headersFrame.headers(),
383                     headersFrame.padding(), headersFrame.isEndStream());
384         } else {
385             Future<Void> future = initializeNewStream(ctx, (DefaultHttp2FrameStream) headersFrame.stream());
386             if (future == null) {
387                 final int streamId = headersFrame.stream().id();
388 
389                 future = encoder().writeHeaders(ctx, streamId, headersFrame.headers(), headersFrame.padding(),
390                         headersFrame.isEndStream());
391 
392                 if (!future.isDone()) {
393                     numBufferedStreams++;
394                     // Clean up the stream being initialized if writing the headers fails and also
395                     // decrement the number of buffered streams.
396                     future.addListener(channelFuture -> {
397                         numBufferedStreams--;
398 
399                         handleHeaderFuture(channelFuture, streamId);
400                     });
401                 } else {
402                     handleHeaderFuture(future, streamId);
403                 }
404             }
405             return future;
406         }
407     }
408 
409     private Future<Void> writePushPromise(final ChannelHandlerContext ctx, Http2PushPromiseFrame pushPromiseFrame) {
410         if (isStreamIdValid(pushPromiseFrame.pushStream().id())) {
411             return encoder().writePushPromise(ctx, pushPromiseFrame.stream().id(), pushPromiseFrame.pushStream().id(),
412                     pushPromiseFrame.http2Headers(), pushPromiseFrame.padding());
413         } else {
414             Future<Void> future = initializeNewStream(ctx, (DefaultHttp2FrameStream) pushPromiseFrame.pushStream());
415             if (future == null) {
416                 final int streamId = pushPromiseFrame.stream().id();
417                 future = encoder().writePushPromise(ctx, streamId, pushPromiseFrame.pushStream().id(),
418                         pushPromiseFrame.http2Headers(), pushPromiseFrame.padding());
419 
420                 if (future.isDone()) {
421                     handleHeaderFuture(future, streamId);
422                 } else {
423                     numBufferedStreams++;
424                     // Clean up the stream being initialized if writing the headers fails and also
425                     // decrement the number of buffered streams.
426                     future.addListener(f -> {
427                         numBufferedStreams--;
428                         handleHeaderFuture(f, streamId);
429                     });
430                 }
431                 return future;
432             }
433             return future;
434         }
435     }

437     private Future<Void> initializeNewStream(ChannelHandlerContext ctx, DefaultHttp2FrameStream http2FrameStream) {
438         final Http2Connection connection = connection();
439         final int streamId = connection.local().incrementAndGetNextStreamId();
440         if (streamId < 0) {
441             Future<Void> f = ctx.newFailedFuture(new Http2NoMoreStreamIdsException());
442 
443             // Simulate a GOAWAY being received due to stream exhaustion on this connection. We use the maximum
444             // valid stream ID for the current peer.
445             onHttp2Frame(ctx, new DefaultHttp2GoAwayFrame(connection.isServer() ? Integer.MAX_VALUE :
446                     Integer.MAX_VALUE - 1, NO_ERROR.code(),
447                     writeAscii(ctx.alloc(), "Stream IDs exhausted on local stream creation")));
448 
449             return f;
450         }
451         http2FrameStream.id = streamId;
452 
453         // Use a Map to store all pending streams as we may have multiple. This is needed as if we would store the
454         // stream in a field directly we may override the stored field before onStreamAdded(...) was called
455         // and so not correctly set the property for the buffered stream.
456         //
457         // See https://github.com/netty/netty/issues/8692
458         Object old = frameStreamToInitializeMap.put(streamId, http2FrameStream);
459 
460         // We should not re-use ids.
461         assert old == null;
462         return null;
463     }
464 
465     private void handleHeaderFuture(Future<?> channelFuture, int streamId) {
466         if (channelFuture.isFailed()) {
467             frameStreamToInitializeMap.remove(streamId);
468         }
469     }

471     private void onStreamActive0(Http2Stream stream) {
472         if (stream.id() != HTTP_UPGRADE_STREAM_ID &&
473                 connection().local().isValidStreamId(stream.id())) {
474             return;
475         }
476 
477         DefaultHttp2FrameStream stream2 = newStream().setStreamAndProperty(streamKey, stream);
478         onHttp2StreamStateChanged(ctx, stream2);
479     }

514     @Override
515     protected void onConnectionError(
516             ChannelHandlerContext ctx, boolean outbound, Throwable cause, Http2Exception http2Ex) {
517         if (!outbound) {
518             // allow the user to handle it first in the pipeline, and then automatically clean up.
519             // If this is not desired behavior the user can override this method.
520             //
521             // We only forward non outbound errors as outbound errors will already be reflected by failing the promise.
522             ctx.fireExceptionCaught(cause);
523         }
524         super.onConnectionError(ctx, outbound, cause, http2Ex);
525     }

527     /**
528      * Exceptions for unknown streams, that is streams that have no {@link Http2FrameStream} object attached
529      * are simply logged and replied to by sending a RST_STREAM frame.
530      */
531     @Override
532     protected final void onStreamError(ChannelHandlerContext ctx, boolean outbound, Throwable cause,
533                                        Http2Exception.StreamException streamException) {
534         int streamId = streamException.streamId();
535         Http2Stream connectionStream = connection().stream(streamId);
536         if (connectionStream == null) {
537             onHttp2UnknownStreamError(ctx, cause, streamException);
538             // Write a RST_STREAM
539             super.onStreamError(ctx, outbound, cause, streamException);
540             return;
541         }
542 
543         Http2FrameStream stream = connectionStream.getProperty(streamKey);
544         if (stream == null) {
545             LOG.warn("Stream exception thrown without stream object attached.", cause);
546             // Write a RST_STREAM
547             super.onStreamError(ctx, outbound, cause, streamException);
548             return;
549         }
550 
551         if (!outbound) {
552             // We only forward non outbound errors as outbound errors will already be reflected by failing the promise.
553             onHttp2FrameStreamException(ctx, new Http2FrameStreamException(stream, streamException.error(), cause));
554         }
555     }

557     private static void onHttp2UnknownStreamError(@SuppressWarnings("unused") ChannelHandlerContext ctx,
558             Throwable cause, Http2Exception.StreamException streamException) {
559         // We log here for debugging purposes. This exception will be propagated to the upper layers through other ways:
560         // - fireExceptionCaught
561         // - fireUserEventTriggered(Http2ResetFrame), see Http2MultiplexHandler#channelRead(...)
562         // - by failing write promise
563         // Receiver of the error is responsible for correct handling of this exception.
564         LOG.log(DEBUG, "Stream exception thrown for unknown stream {}.", streamException.streamId(), cause);
565     }

669         private Http2FrameStream requireStream(int streamId) {
670             Http2FrameStream stream = connection().stream(streamId).getProperty(streamKey);
671             if (stream == null) {
672                 throw new IllegalStateException("Stream object required for identifier: " + streamId);
673             }
674             return stream;
675         }

707     /**
708      * {@link Http2FrameStream} implementation.
709      */
710     // TODO(buchgr): Merge Http2FrameStream and Http2Stream.
711     static class DefaultHttp2FrameStream implements Http2FrameStream {
712 
713         private volatile int id = -1;
714         private volatile Http2Stream stream;
715 
716         final Http2FrameStreamEvent stateChanged = Http2FrameStreamEvent.stateChanged(this);
717         final Http2FrameStreamEvent writabilityChanged = Http2FrameStreamEvent.writabilityChanged(this);
718 
719         Channel attachment;
720 
721         DefaultHttp2FrameStream setStreamAndProperty(PropertyKey streamKey, Http2Stream stream) {
722             assert id == -1 || stream.id() == id;
723             this.stream = stream;
724             stream.setProperty(streamKey, this);
725             return this;
726         }
727 
728         @Override
729         public int id() {
730             Http2Stream stream = this.stream;
731             return stream == null ? id : stream.id();
732         }
733 
734         @Override
735         public State state() {
736             Http2Stream stream = this.stream;
737             return stream == null ? State.IDLE : stream.state();
738         }
739 
740         @Override
741         public String toString() {
742             return String.valueOf(id());
743         }
